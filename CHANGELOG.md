# Changelog

All notable changes to Mneme are documented here.

## 0.1.0 — 2026-07-20

### Added

- Append-only hash-chained semantic event and round-receipt ledgers.
- Typed rationale, provenance, privacy, authority, and supersession contracts.
- Deterministic current-state reduction with unresolved conflict records.
- Bounded context capsules and a context-risk sentinel.
- User-authorized, provenance-bearing handover compilation.
- Capability registry and manifests for ChatGPT Web, Gemini Web, Grok Web, Codex, Claude Code, Antigravity, OpenClaw, Cursor, Pi, and OpenCode.
- Shared browser-companion and native-adapter architecture.
- CLI, JSON schemas, examples, CI, and permanent tests.
